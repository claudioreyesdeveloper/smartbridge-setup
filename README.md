# SmartBridge Setup

SmartBridge Setup helps you install SmartBridge on your computer.

SmartBridge connects your Yamaha keyboard with music tools like Cubase and Synthesizer V.

SmartBridge currently supports Yamaha **Tyros 1, 2, 3, 4, and 5** and **Genos 1 and 2**.

**Download here:** [Latest SmartBridge Setup release](../../releases/latest)

---

## Which File Should I Download?

There are only two choices for this first release.

### If you received a serial number

Download the **Beta 0.1** version.

Use this if Claudio sent you an email with a serial number.

For Mac:

`SmartBridge_Setup_Beta-0.1_0.1.0_arm64.dmg`

For Windows:

`SmartBridge_Setup_Beta-0.1_0.1.0_x64-setup.exe`

### If you only want to try SmartBridge

Download the **Demo** version.

The Demo version works for 30 days.

For Mac:

`SmartBridge_Setup_Demo_0.1.0_arm64.dmg`

For Windows:

`SmartBridge_Setup_Demo_0.1.0_x64-setup.exe`

---

## Mac or Windows?

If you are on a recent Apple Silicon Mac, download the `.dmg` file.

If you are on Windows, download the `.exe` file.

You can also ignore the `Source code` files GitHub adds at the bottom of the release page.

---

## How To Install On Mac

1. Download the `.dmg` file.
2. Open the downloaded file.
3. Drag **SmartBridge Setup** into your Applications folder.
4. Open **SmartBridge Setup**.
5. If your Mac says it cannot open the app, right-click the app and choose **Open**.
6. When SmartBridge Setup opens, click the buttons on the screen to install what you need.

---

## How To Install On Windows

1. Download the `.exe` file.
2. Double-click it.
3. If Windows shows a warning, choose **More info**, then **Run anyway**.
4. Follow the installer.
5. Open **SmartBridge Setup** from the Start Menu.
6. Click the buttons on the screen to install what you need.

---

## Where Do I Enter My Serial Number?

If you downloaded the **Beta 0.1** version, open SmartBridge Setup.

On the main screen you will see a **License** box.

Click **Enter serial**.

Type:

- the email address Claudio used for your beta invitation
- the serial number from that email

Then click **Activate**.

If you typed something wrong, you can try again.

If you need to change the serial later, open SmartBridge Setup again and click **Change serial** in the License box.

---

## What SmartBridge Setup Installs

SmartBridge Setup can install these parts for you:

- the main SmartBridge app and plugin
- the Cubase connection
- the Synthesizer V connection
- help files
- SmartBridge resources
- optional AI Lyrics support

You do not need to understand all of these. SmartBridge Setup checks your computer and shows what is ready, missing, or needs repair.

The Demo version is for trying SmartBridge.
